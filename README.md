## 大带宽 VPS 是什么，挑之前先搞清楚这件事

**大带宽 VPS** 指的是端口带宽峰值在 1Gbps 及以上的虚拟服务器，配合足量的月度流量配额，用来跑高并发、做流量转发、搭建下载节点或承载大流量业务。

问题是，带宽标称值和实际可用是两回事。端口 10Gbps，但路由绕了半个地球，对国内用户来说速度还是跑不起来。

DMIT 解决这个问题的方式很直接：直接拿手里的 CN2 GIA 带宽资源做产品，电信、联通、移动三网回程各走各的优化线路，不在这里凑合。

## 大带宽 VPS 买完流量用超了怎么办

这个问题被问得挺多。DMIT 的处理方式比较干净利索：**超量限速不停机**。

具体来说：

- 入门款（WEE、CORONA 等规格）超量后限速到 2Mbps
- Pro.TINY、Pro.Pocket、EB.TINY 等规格超量后限速到 4Mbps
- STARTER 及以上规格超量后限速到 10Mbps

限速期间机器还在跑，不会直接给你停掉。对于跑任务的场景，这个策略比直接断流要友好很多。

## 四步选对你的大带宽 VPS

1. **先确定目标用户在哪**：主要给国内用户访问，首选香港 Pro 或洛杉矶 Pro；主要跑出境或国际流量，Tier 1 性价比最高
2. **确认运营商**：电信用户走 CN2 GIA 体感最好；联通移动用户 Eyeball 系列已经够用，还便宜一档
3. **估算月流量**：Eyeball 系列同价位流量比 Pro 多，流量敏感的优先看 EB 系列
4. **按付款周期算折扣**：年付比月付省 20-30%，季付也有折扣，有现金流压力就先月付测试

我自己用下来感受是：洛杉矶 EB 系列的性价比在这个价位段确实挺突出，联通移动跑起来不输同价位很多机器，流量还多。电信用户如果对回程线路质量要求高，Pro 系列的 CN2 GIA 双向优化就值那个溢价。

## 常见问题

**Q：DMIT 的 IP 被封了怎么办？**  
A：洛杉矶 Pro 和 EB 系列（不含 sPro）支持免费换 IP，条件是购买满 7 天、距上次换 IP 超过 15 天、全端口被封且剩余有效期超 7 天。不满足条件的情况花 $5 也能换，间隔至少 7 天一次。

**Q：大带宽 VPS 支持用支付宝付款吗？**  
A：支持。DMIT 接受支付宝、PayPal、信用卡，国内用户付款没门槛。

**Q：DMIT 提供 Windows 系统吗？**  
A：不提供。全系只有 Linux 系统，有 Windows 需求的要看其他商家。

**Q：新手购买 DMIT，用 SSH 密钥还是密码登录？**  
A：DMIT 默认用 SSH 密钥登录，购买成功后在控制面板下载私钥文件，密钥只能下载一次，注意保存。第一次没下载到可以重新生成，但原来的密钥就失效了。

**Q：Tier 1 和 Eyeball 主要区别在哪？**  
A：T1 是国际线路，没有针对中国大陆的路由优化，流量按出入站取最大值单向计量；Eyeball 在 T1 基础上增加了国内回程优化（CMIN2、AS9929），流量双向计算，适合有国内访问需求的场景。
